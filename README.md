## Tratamento de Exceções - Java

Repositório dedicado à prática de exceções personalizadas em Java, simulando um sistema de saques bancários.

## Objetivos
* Implementar exceções de domínio (Domain Exceptions).
* Praticar a estrutura try-catch-finally.
* Utilizar os comandos throw e throws.
* Entender a propagação da Stack Trace.

## Tecnologias
* Java
* Git/GitHub
* Ambiente Linux (Ubuntu)
